# Sistema Bancário em Java (POO)

Este projeto é um sistema bancário simples, sem banco de dados, desenvolvido em Java com o objetivo
de praticar os conceitos básicos de Programação Orientada a Objetos (POO).

## Funcionalidades (implementadas até o momento)

- Cadastro de clientes
- Criação de contas bancárias
- Associação de cliente a uma conta
- Fazer deposito de um valor
- Sacar um valor
- Fazer um PIX de uma conta para outra
- Verificações básicas para o programa funcionar e não haver erros
- Historico de saques, depositos e transferencias com PIX

## Como executar

1. Clone o repositório
2. Abra o projeto em uma IDE
3. navegue até controle_financeiro: src\main\java\com\mycompany\controle_financeiro
4. Execute a classe principal (App.java)
