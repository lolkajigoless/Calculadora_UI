# Calculadora com interface gráfica

Este projeto é uma Calculadora desenvolvido em Java com o objetivo
de praticar os conceitos básicos da biblioteca Java Swing.

**Desenvolvido para um trabalho escolar.**

## Funcionalidades

- Somar
- Subtrair
- Multiplicar
- dividir
- se escrever numeros decimais com vírgula ao invés de ponto, o programa ira substituir internamente a virgula por ponto
- Se o divisor for 0 o programa não permite o calculo
- Disponibilização dos temas "Claro" e "Noite"
- Botão de limpar campos

## Como executar

1. Clone o repositório
2. Abra o projeto "CalcUI" em uma IDE
3. navegue até calcui: src\calcui
4. Execute a classe principal (CalcUImain.java)
